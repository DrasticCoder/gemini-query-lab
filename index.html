Gemini Query Lab

Welcome to Gemini Query Lab

Follow these steps to get your own Gemini API Key:

  1. Go to your Google Cloud Console.
  2. Create or select a project.
  3. Enable the Gemini API.
  4. Navigate to the Credentials page and create an API key.
  5. Copy your API key.

How to use this website:

  • Click “Get Started”
  • Enter your Gemini API Key
  • Start prompting and receive answers!

Enter your Gemini API Key

To obtain your API key:

  1. Visit the Google Cloud Console.
  2. Create or select your project.
  3. Enable the Gemini API.
  4. Go to the Credentials section and create an API key.
  5. Paste your API key below.
